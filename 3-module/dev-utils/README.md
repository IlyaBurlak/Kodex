# `ilyaburlak_dev-kit`

Пакет предназначен для упрощения работы с валидацие полей паролей и email

##  Установка
```bash
 npm install ilyaburlak_dev-kit
```

##  Настройка

Для настройки валидации паролей и email нужно запустить команду

```bash
  npx settings
```

## Тестирование
Запуск тестов

```bash
  npm test
```
